# TelecomLab

Aplicación educativa para la asignatura Infraestructuras y Redes de Comunicación (IRC) de la UGR.

## Módulos incluidos

- **GPON Planner**: diseño de redes ópticas pasivas.
- **Mobile Planner**: análisis de interferencia cocanal en redes celulares.
- **Queue Simulator**: simulación de colas M/M/1.
- **ICT Designer**: diseño de redes de distribución de TDT según RD 346/2011.
